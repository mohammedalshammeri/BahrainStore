## 🖥️ الواجهة الأمامية (Frontend)

### لوحة تحكم التاجر (Dashboard)
| التقنية | الاستخدام |
|---------|-----------|
| **Next.js 14** (App Router) | الإطار الرئيسي |
| **TypeScript** | لغة البرمجة |
| **Tailwind CSS** | التصميم |
| **shadcn/ui** | مكونات UI |
| **Zustand** | إدارة الحالة |
| **React Query (TanStack)** | إدارة البيانات |
| **React Hook Form + Zod** | النماذج والتحقق |

### متجر العميل (Storefront)
| التقنية | الاستخدام |
|---------|-----------|
| **Next.js 14** | SSR/SSG لأداء ممتاز |
| **Tailwind CSS** | تصميم متجاوب |
| **Framer Motion** | الحركات والتأثيرات |
| **next-intl** | دعم عربي/إنجليزي |
